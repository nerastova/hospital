# Исследовательский анализ ФГБОУ ВО СПбГПМУ Минздрава России (Педиатрический Университет)

## Данные

Данные о госпитализациях в ФГБОУ ВО СПбГПМУ Минздрава России (Педиатрический Университет)

## Задачи

- предобработка данных;
- исследование факторов влияющих на время с момента получения направления до госпитализации;
- исследование диагнозов;
- проверка гипотез;
- формулирование общего вывода, рекомендаций.

## Цели исследования:

Определить какие факторы влияют на время, которое проходит от выдачи направления пациенту до его поступления в медицинский центр.
